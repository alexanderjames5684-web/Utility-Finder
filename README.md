# Utility Finder 🔍

A Python command-line tool that helps you find the cheapest utility providers in your area based on your ZIP code.

## What it does

- Compares electricity, gas, and water rates by ZIP code
- Shows estimated monthly costs based on your usage
- Tells you which provider is cheapest for each utility

## How to run it

Make sure you have Python installed, then:

```
python utility_finder.py
```

## Requirements

- Python 3.x
- No extra libraries needed

## Notes

This is a work in progress. Rates are estimates based on publicly available data and may not reflect your exact bill.
